# Leitner Box Learning System

Welcome to the Leitner Box Learning System project.

## Documentation
Please refer to [Project Documentation](docs/project_doc.md) for detailed information about the project, roadmap, and design.

## Database Design
The database design is available in [DBML format](docs/database_design.dbml).
The SQL schema can be found in [app/db/schema.sql](app/db/schema.sql).

## Setup
1. Create a virtual environment: `python -m venv venv`
2. Activate it.
3. Install dependencies: `pip install -r requirements.txt`
4. Copy `.env.example` to `.env` and configure your database.
